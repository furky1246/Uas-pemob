# Sambung Ayat Mobile

Aplikasi Android untuk membantu pengguna melatih hafalan Al-Qur'an melalui permainan sambung ayat.

## Anggota Kelompok

* Shidiq
* Rizky
* Rani
* Dimas Bain Sentosa

## Fitur

* Login dan Register
* Dashboard
* Daftar Juz
* Daftar Surah
* Detail Surah
* Audio Ayat
* Game Sambung Ayat (Drag & Drop)
* Score dan Streak
* Progress Belajar
* Profil Pengguna

## Teknologi

### Frontend

* Kotlin Android
* RecyclerView
* Retrofit
* Material Design

### Backend

* PHP
* MySQL
* XAMPP

## Cara Menjalankan

### Backend

1. Jalankan Apache dan MySQL pada XAMPP
2. Import database
3. Letakkan folder API di `htdocs`

### Android

1. Buka project di Android Studio
2. Sync Gradle
3. Jalankan aplikasi

## Struktur Singkat
